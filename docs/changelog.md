# SPAR Engine Changelog

This document tracks changes to the SPAR Engine, adapters, and tooling.

## v0.1 (2025-12-22)
- Initial release of SPAR Engine v0.1
- Core engine with truncated heavy-tail severity sampling
- Scene-based cutoff logic with multiple resolution modes
- Tag-based content filtering and cooldowns
- CLI interface for testing and generation
- Streamlit UI for rapid prototyping and parameter exploration
- Starter content pack with 12 encounter complications
- Full test suite and organized documentation
- Public GitHub repository: https://github.com/joey-carlson/spar-engine
- Installation support for virtual environments and pipx

## Future Versions
- v0.2: State mutation helpers, improved cutoff fiction, expanded content
- v1.0: Stable API, first system adapters, React web interface
